# TypeScript

> TypeScript adds static type definitions to JavaScript. TypeScript types provide a way to describe the shape of an object, providing better documentation, validation and code exploration.

We are currently only using TypeScript in combination with developing Solidity, as Solidity is a typed language it simplifies writing tests and tasks for Solidity contracts.

For web-components we still write them in plain JavaScript.
