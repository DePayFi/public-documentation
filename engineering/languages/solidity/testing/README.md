# Testing in Solidity

## Stack

- [Hardhat (Ethereum development environment)](https://hardhat.org/)
- [Waffle (Smart Contract Testing Framework)](https://getwaffle.io/)
- [TypeScript (Strongly typed JavaScript)](https://www.typescriptlang.org/)
- [TypeChain (TypeScript bindings for Smart Contracts)](https://github.com/ethereum-ts/TypeChain)
- [Ethers.js (JavaScript Ethereum library)](https://docs.ethers.io/)
- [Mocha (JavaScript Test Framework)](https://mochajs.org/)
- [Chai (JavaScript Assertion Library)](https://www.chaijs.com/)
