# Programming Languages

> A formal language comprising a set of instructions that produce various kinds of output. Programming languages are used in computer programming to implement algorithms.

## Focus and limitation

We develop software using only the programming languages TypeScript, Ruby (on Rails), and Solidity,  
as well as Cascading Stylsheets and Markup Language,
because we believe that this toolset is good enough to build great applications.

We also know that adding any additional language to that landscape
will always come with an overhead we do not take lightly.
