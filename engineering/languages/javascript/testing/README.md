# Testing in JavaScript

## Stack

- [Cypress (Front end testing tool built for the modern web)](https://docs.cypress.io/guides/overview/why-cypress.html#In-a-nutshell)
- [TypeScript (Strongly typed JavaScript)](https://www.typescriptlang.org/)
- [Mocha (JavaScript Test Framework)](https://mochajs.org/)
- [Chai (JavaScript Assertion Library)](https://www.chaijs.com/)

## Cypress

### Test single file

```
yarn test --spec "cypress/integration/my-spec.js"
```

### Show interactive browser

```
yarn test --headed
```
