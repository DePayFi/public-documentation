# Software Licensing

## Closed Source

All our closed source software,
especially everything published within the DePay PRO ecosystem,
is proprietary software and NOT open source.

## Open Source

For all our open source software we issue MIT license (NOT GPL),
because we do not want to force people that use our software to also publish theirs.
